# Angola Hoje

Angola Hoje é um portal web desenvolvido para fornecer informações essenciais e atualizadas sobre Angola. A plataforma reúne dados importantes como taxas de câmbio, preços de combustível, notícias relevantes e oportunidades de emprego, tudo em um único lugar.

O projeto foi criado com o objetivo de facilitar o acesso à informação para cidadãos, profissionais e empreendedores que precisam acompanhar dados económicos e oportunidades no país.

---

## Funcionalidades

- Exibição de taxas de câmbio (USD e EUR)
- Informações sobre preços de combustível
- Notícias atualizadas através de API
- Lista de vagas de emprego e concursos
- Sistema de alertas por e-mail
- Geração de relatórios em PDF
- Interface responsiva para telemóveis e computadores

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- APIs externas (notícias e câmbio)
- Biblioteca jsPDF para geração de documentos
